HOW IT WORKS:
───────────────────────────────────────────────────────────────

✓ Backend API unavailable?
  → System automatically falls back to LOCAL AUTHENTICATION
  → Demo accounts are pre-loaded and ready
  → Portfolio data is generated mock-data (stored in localStorage)

✓ Can I create new accounts?
  → YES! Click "Create Account" and use the registration form
  → New accounts are stored in localStorage (browser-based)
  → Your data persists across page reloads

✓ Is this secure?
  → Demo accounts are for TESTING ONLY
  → Passwords are stored in plaintext (demo mode only!)
  → DO NOT use real passwords or copy to production
  → This is a fully functional local testing system
